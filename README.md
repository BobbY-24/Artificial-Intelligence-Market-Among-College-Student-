# AI Assistant Usage Among College Students

## Overview
This project analyzes survey-style data about AI assistant usage in student life. It explores how college students use AI tools, what patterns appear in the dataset, and what kinds of questions could be studied with more rigorous survey design. The project is an exploratory data analysis notebook rather than a causal study.

## Motivation
AI assistant adoption is an important topic for education, productivity, and student behavior. This project connects directly to broader interests in human-AI interaction and responsible AI use. It demonstrates exploratory analysis, data cleaning, and the ability to turn a current social/technical question into a structured dataset investigation.
